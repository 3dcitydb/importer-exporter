This document is the API specification for the 3D City Database Importer/Exporter Plugin API 1.4-devel.
The 3D City Database Importer/Exporter Plugin API is a Java class library and API facilitating the development of own extensions for the 3D City Database Importer/Exporter program.
The 3D City Database Importer/Exporter Plugin API is open source and released under the terms of the GNU Lesser General Public License v3 (LGPL).
