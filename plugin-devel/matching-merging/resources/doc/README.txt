3. About
--------

The !plugin.match.name.full! 
is an extension of the !impexp.name! program.
It allows for identifying building objects within the database which
occupy the same spatial area (partially or completely). Candidate
matches can be merged into a single building object. 


4. Requirements
---------------

This version of the !plugin.match.name! shall only be used with
version !impexp.version! of the !impexp.name!. 
It cannot be guaranteed that the extension does correctly work with any
previous or later version of the !impexp.name!.


5. (Un)Installation
-------------------

The installation of the !plugin.match.name! is automatically 
carried out by the universal installer during the setup procedure
of the !impexp.name! if the corresponding 
installation package has been chosen by the user.

The plugin is shipped as a precompiled and ready-to-use JAR package
called "!plugin.match.jar.filename!". The universal installer will put this JAR
package into the "!dir.dest.plugins.relative!/!plugin.match.dir.dest.relative!" subfolder
within the installation folder of the !impexp.name!.

If you want to uninstall the !plugin.match.name!, simply delete
the "!plugin.match.dir.dest.relative!" subfolder and all its
contents.
